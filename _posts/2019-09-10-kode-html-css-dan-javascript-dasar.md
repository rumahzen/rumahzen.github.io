# HTML

## Menampilkan Gambar

```html
<img src='https://situs.com/gambar.jpg'>
```

## Membuat Link

```html
<a href='https://situs.com'>klik di sini</a>
```

## Import CSS dari Luar

```html
<link rel='stylesheet' href='https://situs.com/style.css'>
```

## CSS dalam HTML

```html
<style>
	img {
		max-width: 100%;
	}
</style>
```

## Tebal

```html
<strong>teks</strong>
```

## Miring

```html
<em>teks</em>
```

## Ordered List

```html
<ol>
	<li>satu</li>
	<li>dua</li>
</ol>
```

## Unordered List

```html
<ul>
	<li>tempe</li>
	<li>tahu</li>
	<li>saos</li>
</ol>
```
