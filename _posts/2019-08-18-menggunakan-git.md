# Mempersiapkan Akun Github

```bash
git config --global user.name username
git config --global user.email email
```

# Clone Repository

```bash
git clone --depth=1 https://username:password@github.com/username/repositori
```

# Upload

```bash
git status
git add -A .
git commit -m "oke"
git push
```

# Update Repositori Local

```bash
git pull
```
